About BatteryData

This battery materials database was auto-generated using ChemDataExtractor.

It contains 292,313 data records with 17,354 unique chemical compound, extracted from 229,061 journal articles

Database generation

ChemDataExtractor uses state-of-the-art natural language processing algorithms to interpret language text that makes up the majority of scientific documents.
Rule-based parsers are used in ChemDataExtractor-bateries version 1.5 for database construction.

Cite: Huang, S., Cole, J.M. A database of battery materials auto-generated using ChemDataExtractor. Sci Data 7, 260 (2020). 10.1038/s41597-020-00602-2
